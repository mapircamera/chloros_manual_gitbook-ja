## 言語を変更する方法

### Chlorosデスクトップ/ブラウザ内

1. アプリケーション設定を開きます
2. 言語選択メニューに移動します
3. リストから希望の言語を選択してください
4. インターフェースは即座に更新されます

### Chloros CLI の場合

コマンドで言語パラメータを使用して、インターフェイス言語を設定します。

### Chloros Python SDK 内

SDK を初期化するときに言語パラメーターを設定して、希望の言語でメッセージと出力を取得します。

## カバレッジ

38 言語すべてが以下で完全にサポートされています。

* **Chloros デスクトップ** - 完全な GUI 翻訳
* **Chloros ブラウザ** - すべての言語の Web インターフェイス
* **Chloros CLI** - コマンドライン インターフェイスと出力メッセージ
* **Chloros Python SDK** - API メッセージとドキュメント

言語サポートにより、世界中のユーザーが障壁なく母国語で効率的に作業できるようになります。
