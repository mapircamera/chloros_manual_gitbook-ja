# 画像グリッド

画像をプロジェクトにインポートすると、メイン領域にグリッド状に配置された画像が表示されます。

## サムネイルサイズ

右上のズームスライダーで画像サムネイルのサイズを調整できます。また、CTRLキーを押しながらマウスホイールを操作しても拡大縮小できます。

## エクスポートビューボタン

画像の処理が進むと、グリッド領域の左上にボタンが表示されます。 これらのボタンでグリッドサムネイルの表示タイプを切り替えられます：

* **JPG** - カメラ元のJPGサムネイル
* **ターゲット** - 検出したキャリブレーションターゲットを含む画像
* **反射率** - キャリブレーション済み反射率画像
* **インデックス/LUT** - カラーLUTグラデーション付き計算済みマルチスペクトルインデックス画像

画像グリッドのサムネイルをクリックすると、フルスクリーン[画像ビューアー](opening-an-image-full-screen.md)が開きます。グリッドの表示モードが「ターゲット」などの特定のエクスポート形式に設定されている場合、グリッドサムネイルをクリックするとエクスポートされたターゲット画像がフルスクリーンで開きます。
