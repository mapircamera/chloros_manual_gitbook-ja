# 画像グリッド

画像をプロジェクトにインポートすると、メインエリアにグリッド状に配置されます。

## サムネイルサイズ

右上のズームスライダーで画像サムネイルのサイズを調整できます。また、CTRLキーを押しながらマウスホイールを操作して拡大縮小することも可能です。

<figure><img src="../.gitbook/assets/chloros_grid_zoom.gif" alt=""><figcaption></figcaption></figure>## エクスポート表示ボタン

画像処理が進むと、グリッド領域の左上にボタンが表示されます。これらのボタンでグリッドサムネイルの表示タイプを切り替えられます：

* **JPG** - カメラ元のJPGサムネイル
* **ターゲット** - 検出したキャリブレーションターゲットを含む画像
* **反射率** - キャリブレーション済み反射率画像
* **RWA (インデックス/LUT)** - カラーLUTグラデーションを適用した計算済みマルチスペクトルインデックス画像

画像グリッドのサムネイルをクリックすると、フルスクリーンで[画像ビューアー](opening-an-image-full-screen.md)が開きます。グリッドの表示モードが「ターゲット」などの特定のエクスポート形式に設定されている場合、グリッドサムネイルをクリックするとエクスポートされたターゲット画像がフルスクリーンで開きます。
