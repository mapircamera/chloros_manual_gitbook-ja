# 画像グリッド

プロジェクトに画像をインポートすると、メインエリアに画像がグリッド状に表示されます。

## サムネイルのサイズ

右上のズームスライダーを使って、画像サムネイルのサイズを調整できます。また、Ctrlキーを押しながらマウスホイールを回すと、サムネイルの拡大・縮小が可能です。

<figure><img src="../.gitbook/assets/chloros_grid_zoom.gif" alt=""><figcaption></figcaption></figure>## エクスポート表示ボタン

画像の処理が進むにつれて、グリッド領域の左上にボタンが表示されます。これらのボタンをクリックすると、グリッド上のサムネイルが表示される画像の種類を切り替えることができます：

* **JPG** - カメラ元のJPGサムネイル
* **ターゲット** - 検出されたキャリブレーションターゲットを含む画像
* **反射率** - キャリブレーション済みの反射率画像
* **RWA (インデックス/LUT)** - カラーLUTグラデーションが適用された、計算済みのマルチスペクトルインデックス画像

画像グリッドのサムネイルをクリックすると、[画像ビューア](opening-an-image-full-screen.md)が全画面で開きます。グリッドの表示モードが「ターゲット」などの特定のエクスポート形式に設定されている場合、グリッドのサムネイルをクリックすると、エクスポートされたターゲット画像が全画面で開きます。
