# インデックス/LUTサンドボックス

インデックス/LUTサンドボックスは、Chloros Image Viewer内に搭載されたインタラクティブなワークスペースであり、マルチスペクトルインデックスの計算やカラー可視化をリアルタイムで試すことができます。この強力なツールを使用すれば、データセット全体を再処理することなく、さまざまなインデックスのテスト、値の範囲の調整、そして出版物としてそのまま使える可視化の作成が可能です。

## インデックス/LUTサンドボックスとは？

### 目的

サンドボックスでは以下の機能を提供します：

* **リアルタイムのインデックス計算** - あらゆる植生インデックスを即座に適用
* **インタラクティブなLUT調整** - 色のグラデーションや範囲を微調整
* **ワークフローの最適化** - バッチ処理の前に最適な設定を決定

### サンドボックスとプロジェクト処理の比較

**インデックス/LUTサンドボックス（インタラクティブ）：**

* 1回につき1枚の画像
* 即座に結果が反映される
* 実験的かつ反復的な作業に適している
* ファイルに永続的な変更を加えない
* 探索やテストに最適

**プロジェクト処理（バッチ）：**

* データセット全体を一括処理
* 事前設定済み
* 出力ファイルは永続的
* 処理に時間がかかる
* 設定が確定した後に最適

{% hint style="success" %}
**最適なワークフロー**：サンドボックスで実験を行い、最適なインデックスとLUTの設定を見つけ、その後、プロジェクト処理でデータセット全体にそれらの設定を適用します。
{% endhint %}

## インデックス/LUTサンドボックスの使用

### 事前計算済みインデックスの理解

Chlorosでは、プロジェクト処理中にインデックスを適用できます。エクスポートに適用するインデックスとLUTの設定を決定するには、画像ビューアーのサンドボックスを使用するのが最も簡単です。

サンドボックスでは、以下の操作が可能です：

* **新しいインデックスやカラーグラデーション（LUT）を適用**してデータを可視化する
* **可視化設定を**インタラクティブに調整する
* **すでに計算済みのインデックス画像を**表示する
* **すべてのズームレベルで**ピクセル値を**確認**する

### サンドボックスの起動

インデックス/LUTサンドボックスは、**画像ビューア**の <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> のサイドバータブからアクセスできます：

1. ファイルブラウザの画像グリッドで画像をクリックすると、**画像ビューア** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> タブで開きます
2. **Image Viewer** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> タブをクリックして、左側のポップアウトサイドバーを開きます（まだ開いていない場合）

### インデックス/LUTを適用する画像の選択

Image Viewer <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> サンドボックスでインデックスを操作するには：

1. メインの画像グリッドから画像をクリックして**開く**

2. すると**Image Viewer** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> タブが開きます
3. **レイヤードロップダウン**（ビューアーの右上）をクリックします
4. ドロップダウンからレイヤーを選択します：
   * RAW (反射率)

### 画像へのインデックスの適用

画像が全画面表示され、**画像ビューアー** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> タブのサイドバーが開いている状態で：

1. サイドバー上部の「インデックス」チェックボックスをオンにします
2. 左側のドロップダウンからカメラのフィルターを選択します
3. 右側のドロップダウンから希望のインデックス式を選択します
4. フィルターチャンネルの色の円を、下のインデックス式内の対応する位置にドラッグします
5. 式が有効になると、画像が更新され、インデックス値が表示されます
6. マウスカーソルを動かして、カーソルの位置の値を確認します
7. ズームインして、個々のピクセルとその関連値を確認します

各インデックスには、固有の値の範囲と意味があります：

#### NDVI の例

```

Formula: (NIR - Red) / (NIR + Red)

For Survey3W RGN camera:
NIR = 850nm band
Red = 661nm band

Result range: -1.0 to +1.0
Typical vegetation: 0.4 to 0.9
Stressed vegetation: 0.2 to 0.4
Bare soil: 0.0 to 0.2
Water: -0.1 to 0.1
```

インデックス式の完全なドキュメントについては、[マルチスペクトルインデックス式](../project-settings/multispectral-index-formulas.md)を参照してください。

## LUT（ルックアップテーブル）の操作

### LUTとは？

**ルックアップテーブル（LUT）**は、視覚化のために数値インデックス値を色にマッピングします：

* **入力**：インデックスピクセル値（例：NDVI 0.65）
* **出力**: 色（例：明るい緑）
* **目的**: パターンをより見やすく、解釈しやすくする**グレースケールとカラーLUTの比較:**

* グレースケール: 科学的かつ中立的、生データを表示
* カラーLUT: 直感的でインパクトがあり、パターンや差異を強調

{% hint style="success" %}
**可視化の威力**: グレースケールのインデックス画像にカラーLUTを適用すると、パターン、異常、注目すべき領域を一目で識別するのが劇的に容易になります。
{% endhint %}

### インデックス画像へのLUTの適用

インデックス画像が表示されたら

1. <img src="../.gitbook/assets/image (1) (1).png" alt="" data-size="line"> 「+LUTを追加」ボタンをクリックします
2. カラーグラデーションを選択します
3. クリッピングの最小/最大端点を調整します
4. クリッピングモードを調整します
5. **画像ビューア** <img src="../.gitbook/assets/icon_image-viewer.JPG" alt="" data-size="line"> タブのサイドバーにある「インデックス」チェックボックスをオンにして、LUTを適用します

### カラーグラデーションの選択

**グラデーションの選択方法：**

1. LUTパネルで、**色付きグラデーションバー**を探します
2. マウスをその上に合わせると、利用可能なグラデーションプリセットが表示されます
3. 希望のグラデーションを選択します
4. インデックスボックスにチェックを入れると、画像は新しい色で**即座に更新**されます

{% hint style="success" %}
**ベストプラクティス**：NDVIのような植生指数の場合、Red-Yellow-Greenのグラデーションが最も直感的です。これは、自然な色の連想（緑＝健全、黄色＝中程度、赤＝ストレス）と一致しているためです。
{% endhint %}

### 色クラスの調整

**「クラス」コントロール**は、グラデーションに表示される離散的な色の段階の数を決定します：**クラス数のオプション：*** **2～5クラス**：非常に大まかな分類、明確なゾーン
* **6～10クラス**：バランスが取れており、分類に適している
* **11～20クラス**：滑らかなグラデーション、連続的な外観
* **20クラス以上**：ほぼ連続的、最大の滑らかさ**調整方法：**

1. LUTパネルで、**グラデーションバーの下にあるカラースウォッチの四角形**を探します
2. **+**ボタンをクリックしてクラスを追加し、数を調整します
3. カラースウォッチをダブルクリックして、クラスを削除します
4. 画像上のグラデーションは**リアルタイム**で更新されます**可視化への影響：*** **クラス数が少ない場合** (3～5): 明確なゾーンが形成され、分類が簡素化され、カテゴリの区別が容易になります
* **中程度のクラス数** (6～10): バランスの取れたアプローチで、ほとんどの用途に適しています
* **クラス数が多い場合** (15～20): 滑らかな遷移、詳細な変化、写真のような外観**使用シーン：*** **クラス数が少ない場合（3～5）**：プレゼンテーション用スライド、分類マップ、簡易レポート
* **クラス数が中程度の場合（6～10）**：一般的な分析、バランスの取れた詳細度、標準的なレポート
* **クラス数が多い場合（15～20）**：科学的な分析、詳細な検査、出版物レベルの品質の出力

### 値の範囲の微調整

**値範囲のコントロール**は、グラデーションにおいてどのインデックス値がどの色に割り当てられるかを決定します：**LUTパネル内の範囲コントロール：*** **最小値**：カラースケールの下限
* **最大値**：カラースケールの上限
* **中間値**：最小値と最大値の間に（クラス数に基づいて）自動的に配分されます

#### 最小値/最大値の調整

**値の範囲を調整するには：**

1. LUTパネルで、**最小値**および**最大値**の入力フィールドを探します
2. **最小値**フィールドをクリックします
3. 希望する最小値を入力します（例：`0.2`）
4. **Enter**キーを押すか、フィールドの外側をクリックします
5. **Max Value**フィールドについても同様に繰り返します（例：`0.9`）
6. 表示が**即座に更新**されます

{% hint style="info" %}
**自動スケーリング**： LUTを初めて適用すると、Chlorosは自動的に最小値/最大値を画像内の実際のデータ範囲に設定します。その後、この範囲を絞り込んで、関心のある特定の値の範囲に焦点を合わせることができます。
{% endhint %}

**範囲調整の例 NDVI：*** **全範囲**: `-1.0` から `1.0` (すべての可能な値を表示)
* **植生に焦点を当てた**: `0.2` から `0.9` （裸地や水域を除外）
* **健全な植生のみ**: `0.5` から `0.9` （活力のある植物のみを強調）
* **ストレス検出**: `0.2` から `0.5` （問題のある領域を強調）
* **カスタム範囲**：観測されたピクセル値に基づいて調整**なぜ範囲を調整するのか？*** 関心領域の**コントラストを高める*** **無関係な値を除外する**（例：水域、裸地）
* 複数の画像や日付にわたって**可視化を標準化する*** 狭い値の範囲内での**微妙な違いを強調する**

### 範囲外の値のクリッピング

ピクセル値が定義した最小/最大範囲外にある場合、**クリッピングモード**を使用してその表示方法を制御できます。

#### **利用可能なクリッピングモードのオプション：**

#### 1. 最小値と最大値

* **最小値未満**のピクセル → グラデーションの**最初の色**（例：赤）を使用して表示
* **最大値**を超えるピクセル → グラデーションの**最後の色**（例：緑）を使用して表示
* **使用例**：極端な値を強調し、限界値で彩度の高い色を使用してデータ範囲全体を表示
* **例**：NDVIの0.2未満の値はすべて赤で表示され、0.9を超える値はすべて緑で表示される

#### 2. 背景の透明化

* **範囲外**のピクセルは**完全に透明**になります
* **範囲内**のピクセルのみがグラデーションで表示されます
* **使用例**: GISオーバーレイ、特定の値の範囲を分離、関心領域のみを強調
* **例**: NDVIの0.4～0.7の範囲のみを色で表示し、それ以外はすべて透明にする

{% hint style="warning" %}
**透明度の制限**：透明なピクセルはビューア上で背景色として表示されます。処理中にエクスポートする場合、PNG形式では透明度が保持されますが、JPG形式では保持されません。
{% endhint %}

#### 3. インデックス背景

* **範囲外**のピクセルは**グレースケール**で表示されます（生のインデックス値が表示されます）
* **範囲内**のピクセルは**カラーグラデーション**で表示されます
* **使用例**: 微妙な強調表示。関心領域を強調しつつ、文脈を維持します
* **例**：ストレスを受けた植生を色で強調（NDVI 0.3-0.5）し、健全な領域はグレーで表示

#### 4. オリジナル背景

* **範囲外**のピクセルは**元のマルチスペクトル画像**を表示
* **範囲内**のピクセルは**色のグラデーション**を表示
* **用途**：最も直感的 - 自然な画像のコンテキストと分析用のカラーオーバーレイを組み合わせる
* **例**：実際の畑や作物の外観を確認しつつ、色分けされたストレス領域をオーバーレイ表示

### 適切なクリッピングモードの選択

| クリッピングモード              | 最適な用途                                   | 視覚化スタイル          |
| -------------------------- | ------------------------------------------ | ---------------------------- |
| **最小値と最大値**    | データ全表示、科学的な分析     | すべてのピクセルに色付け           |
| **透明な背景** | GISオーバーレイ、特定の範囲の抽出    | 範囲内に色、範囲外は白 |
| **インデックス背景**       | 微妙な強調、データコンテキストの維持  | 範囲内に色、範囲外はグレー  |
| **元の背景**    | レポート、プレゼンテーション、直感的な分析 | 範囲内に色、範囲外は写真そのまま |

### カスタムLUTカラーの作成

可視化を完全に制御するために、個々のカラーストップを編集して**カスタムカラーグラデーション**を作成できます。**カスタムグラデーションを作成するには：**

1. LUTパネルで、**グラデーションプレビューバー**を見つけます
2. グラデーションの下にある**カラースウォッチの四角形**を探します
3. **カラーストップをクリック**して選択します
4. **カラーピッカー**が開きます
5. 以下の方法を使用して新しい色を選択します：
   * **カラーホイール**：視覚的な色選択
   * **RGB/HSVスライダー**：精密な色調整
   * **16進数コードの入力**：正確な色指定（例：赤の場合は`#FF0000`）
6. カラーピッカーの外側をクリックして**新しい色を適用**します
7. 画像上のグラデーションが**即座に更新**されます**カラーストップの追加または削除：*** **ストップを追加**：+ アイコンをクリックして、最後に新しいスウォッチを追加します
* **ストップを削除**：色の四角をダブルクリックして、スウォッチを削除します**カスタマイズの手法：*** **グラデーションの反転**：色の順序を入れ替えて意味を逆転させる（例：緑＝低、赤＝高）
* **ブランドカラー**：レポートに組織のカラーパレットを合わせる
* **色覚障害者に配慮**：オレンジと青、または紫と黄色の組み合わせを使用する
* **印刷の最適化**：カラー印刷とグレースケール印刷の両方で適切に表示される色を選択します
* **マルチしきい値**：分類のために、特定の値のしきい値ごとに異なる色を使用します

{% hint style="info" %}
**カスタムグラデーションの保存**：カスタムグラデーションは保存して再利用できます。LUTパネルの保存アイコンをクリックして、カスタム配色を保存し、後で再利用できるようにします。
{% endhint %}

## インタラクティブなワークフロー

### リアルタイム更新

サンドボックス内でのすべてのLUT調整は、画像を**瞬時にかつインタラクティブに**更新します：

* **レイヤーの切り替え** → 画像が即座に変化
* **グラデーションの選択** → 色が瞬時に更新
* **値の範囲の調整** → コントラストがリアルタイムで変化
* **クラスの変更** → グラデーションの滑らかさが即座に更新されます
* **クリッピングの変更** → 背景の表示が即座に変わります
* **色の編集** → カスタムグラデーションが即座に適用されます**「適用」ボタンは不要** - すべての変更はライブかつインタラクティブです！

{% hint style="success" %}
**ライブフィードバック**：瞬時の視覚的フィードバックにより、分析ニーズに最適な可視化が見つかるまで、さまざまな設定を迅速に試すことができます。
{% endhint %}

### 反復的な微調整ワークフロー

**一般的なLUT最適化ワークフロー：**

1.**インデックスレイヤーを選択**（例：RAW（反射率））
2. **インデックスを適用** - カメラフィルターとインデックス式を選択し、色付きの円をインデックス式内の適切な位置にドラッグします
3. **LUTグラデーションを適用** - Red-Yellow-Greenプリセットから開始します
4. **ピクセル値の確認** - カーソルを動かして、値の範囲を確認する
5. **最小/最大値の調整** - 植生に焦点を絞るために範囲を狭める（例：0.2～0.9）
6. **クリッピングの選択** - 背景の文脈を確認するために「Original Background」を試す
7. **色の微調整** - 特定の強調が必要な場合は、グラデーションをカスタマイズする
8. **設定の確定** - 設定内容を記録し、エクスポート処理のためにプロジェクト設定にコピーする

### ピクセル値の確認

効果的なLUT範囲を設定するには、実際のピクセル値を理解することが不可欠です：

**値の確認方法：**

1. 画像の「インデックス」または「インデックス」と「LUT」の両方の**チェックボックスがオン**になっている場合、ピクセル値が表示されます。
2. **カーソルを**画像のさまざまな領域に合わせます
3. カーソルを合わせると、凡例に**ピクセル値**が表示されます
4. ズームインして、浮動値でハイライトされた個々のピクセルを確認します
5. 異なる特徴ごとの値の範囲を**メモ**します：
   * **健全な植生**：例：NDVI 0.55-0.85
   * **ストレスを受けた植生**：例：NDVI 0.30-0.50
   * **裸地**：例：NDVI 0.05-0.25
   * **水**（存在する場合）：例：NDVI -0.05～0.10**ピクセル値を用いたLUT範囲の設定：**ピクセル値を確認した後、それに応じてLUTの最小値/最大値を調整してください：**シナリオ例：*** **観測結果**：土壌値 = 0.05-0.25、ストレス状態 = 0.25-0.50、健全状態 = 0.50-0.85
* **目的**：植物の健康状態のみを可視化（土壌を除外）
* **LUT設定**：Min = `0.25`、Max = `0.85`
* **クリッピング**：「Original Background」を選択し、土壌を自然な色で表示
* **結果**：色のグラデーションは植生にのみ適用され、土壌は元の画像のまま表示される

{% hint style="info" %}
**ダイナミックレンジ**：作物、季節、生育段階によって値の範囲は異なります。LUTの範囲を設定する前に、必ず対象のデータセットのピクセル値を確認してください。
{% endhint %}

## カスタムインデックス (Chloros+)

### カスタムインデックス式の作成

{% hint style="info" %}
**作成場所**: カスタムインデックスは、処理前の**プロジェクト設定**およびイメージビューアのサンドボックスサイドバーで設定できます。
{% endhint %}

**カスタムインデックスの作成方法：**

1.**プロジェクト設定**（処理前）またはイメージビューアーのサンドボックスサイドバーを開きます
2. **インデックス式ドロップダウン**に移動します
3. **「カスタム」**オプションを探します（Chloros+ライセンスでログインしている必要があります）
4. バンド変数を使用して**式を定義**します：
   * バンド名：`NIR`、`Red`、`Green`、`Blue`、`RedEdge` など
   * 演算子: `+`, `-`, `*`, `/`, `^` (指数)
   * 関数: `sqrt()`、`abs()` など（サポートされている場合）
   * 括弧: 演算順序を指定するための `()`
5. **インデックスに名前を付ける**（例: 「MyIndex」または「CustomNDVI」）
6. **設定を保存**

**カスタム数式の例：**

```

Modified NDVI with offset:
(NIR - Red) / (NIR + Red + 0.5)

Simple ratio:
NIR / Red

Complex multi-band:
(NIR - Red) / (NIR + Red - Blue)

Exponential index:
(NIR / Red) ^ 2
```

{% hint style="warning" %}
**数式の検証**：数式に、お使いのカメラで利用可能なバンドが使用されていることを確認してください。例えば、RedEdge は、RedEdge フィルターを搭載したカメラでのみ利用可能です。
{% endhint %}
