# マップマーカー

「マップ」タブでは、画像のGPS座標に基づいて、インタラクティブな2Dマップ上に画像が表示されます。これにより、撮影セッションの地理的な概要を把握でき、撮影範囲を視覚的に確認するのに役立ちます。また、画像を最初にインポートする際、処理する必要のない画像を素早く除外するのにも便利です。

<figure><img src="../.gitbook/assets/chloros_map_markers.gif" alt=""><figcaption></figcaption></figure>

## 「マップ」タブの表示方法

1. Chlorosでプロジェクトを開くか、新規作成します
2. GPSメタデータを含む画像をインポートします
3. 左サイドバーの**「マップ」** <img src="../.gitbook/assets/image (3).png" alt="" data-size="line"> タブをクリックします
4. 地図上に、各画像のGPS位置を示すマーカーが表示されます

{% hint style="info" %}
**GPS必須**：EXIFメタデータにGPS座標が埋め込まれている画像のみが地図上に表示されます。撮影時にカメラのGPS機能が有効になっていることを確認してください。
{% endhint %}

## 画像マーカー

GPSデータを含む各画像は、地図上のマーカーで表示されます：

### マーカーの表示

* マーカーは、各画像が撮影された正確なGPS座標を示します
* ズームアウトすると、密集したマーカーがグループ化される場合があります
* 個々の画像の位置を確認するには、ズームインしてください

{% hint style="success" %}
スーパーズーム：地図タイルプロバイダーから提供される最大ズームレベルに達すると、さらにズームインした際にタイルが拡大され、密集したマーカーを確認できるようになります。
{% endhint %}

### ホバープレビュー

* マーカーの上に**マウスをホバー**させると、その画像のサムネイルプレビューが表示されます
* これにより、地図ビューを離れることなく、素早く視覚的に識別できます
* 大規模な撮影セッション内で特定の画像を探す際に便利です

***

## 地図タイルプロバイダー

{% hint style="success" %}
**自動選択**：Chlorosは、現在の地図位置に対して最適なズームレベルを提供するタイルサービスを自動的に選択します。必要に応じて、手動でプロバイダーを切り替えることも可能です。
{% endhint %}

[地図]タブでは、背景地図画像用に2つのタイルプロバイダーをサポートしています：

### Google Maps

* Google 提供の標準的な衛星画像および地図画像
* 世界中の広範囲をカバーするのに最適

### ESRI

* ESRI ArcGIS 提供の衛星画像および航空写真
* 特定の地域では、より高解像度の画像を提供する場合が多い

## 地図タイルの種類

地図レイヤーの種類を（左から右の順に）選択できます：

 <img src="../.gitbook/assets/image (23).png" alt="" data-size="original">### 地形

標高プロファイルと、詳細（道路など）を含む地図タイルを表示します

### 地図

標準（低帯域幅）の地図タイルと、詳細（道路など）を表示します

### 衛星

詳細（高帯域幅）な衛星地図タイルを表示します

### ハイブリッド

詳細（道路など）が追加された衛星地図タイルを表示します

## 地図の操作

### ズーム操作

* **ズームイン/ズームアウト**：マウスのスクロールホイールまたはズームボタンを使用
* **全画面表示**：地図を全画面表示

### パン操作

* **パン**：クリックしてドラッグし、地図上を移動***

## 活用例

### 飛行経路の可視化

* ドローン撮影セッションの撮影範囲を確認する
* 画像の撮影範囲に空白部分がないか特定する
* 飛行経路の実行を確認する

### 地上調査のレビュー

* 地上からの撮影データの空間的な分布を確認する
* 調査エリアに対するキャリブレーションターゲット画像の位置を特定する
* 追加の撮影場所を計画する

### 品質管理

* 予期しない場所で撮影された画像を迅速に特定する
* データセット全体のGPS精度を確認する
* 画像の位置情報をフィールドノートと照合する

## トラブルシューティング

### マーカーが表示されない

**考えられる原因:**

* 画像にGPSメタデータが含まれていない
* 撮影中にカメラのGPSが無効になっていた
* 外部ソフトウェアによってEXIFデータが削除された

**解決策**: カメラでGPSが有効になっていることを確認し、元のファイルを再インポートしてください

### マーカーの位置が間違っている

**考えられる原因:**

* カメラのGPSが衛星を十分に捕捉できていなかった
* 撮影中にGPSの誤差が生じた

**解決策**: これは通常、撮影時の問題です。高精度な用途では、PPK/RTK GPSの使用をご検討ください
