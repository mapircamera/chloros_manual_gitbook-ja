# 出力画像フォーマット

### TIFF (16ビット)

16ビット (ピクセルデジタル数値 0 - 65535) TIFF (.tif) フォーマット出力。

### TIFF (32ビット、パーセント %)

32ビット (ピクセル浮動小数点 0.0 ～ 1.0) TIFF (.tif) 形式で出力。ピクセル値は反射率のパーセントを表し、0.0 = 0%、1.0 = 100% です。

### **JPG (8ビット)**

8ビット（ピクセルデジタル値 0～255）JPG (.jpg) 形式出力。

### **PNG (8ビット)**

8ビット（ピクセルデジタル値 0 ～ 255）PNG (.png) 形式出力。ロスレス圧縮で、ウェブ閲覧や可視化に適しています。
