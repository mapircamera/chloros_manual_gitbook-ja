# 出力画像形式

### TIFF (16ビット)

16ビット（ピクセル値 0 ～ 65535）のTIFF (.tif) 形式で出力します。

### TIFF (32ビット、パーセント %)

32ビット（ピクセル浮動小数点 0.0 ～ 1.0）の TIFF (.tif) 形式で出力します。ピクセル値は反射率のパーセンテージを表し、0.0 = 0%、1.0 = 100% となります。

### **JPG (8ビット)**

8ビット（ピクセル値：0～255）のJPG (.jpg) 形式で出力。

### **PNG (8ビット)**

8ビット（ピクセル数値 0 ～ 255）のPNG (.png) 形式で出力されます。Webでの閲覧や可視化に適したロスレス圧縮です。
