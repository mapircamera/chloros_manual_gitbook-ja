## 言語の変更方法

### Chloros デスクトップ/ブラウザ版

1. アプリケーション設定を開く
2. 言語選択メニューに移動する
3. リストからお好みの言語を選択する
4. インターフェースが即時更新されます

### Chloros CLI での変更方法

コマンドに言語パラメータを指定してインターフェース言語を設定します。

### Chloros Python SDK での設定

SDK の初期化時に言語パラメータを設定すると、メッセージや出力を希望の言語で表示できます。

## 対応範囲

全38言語が以下で完全対応：

* **Chloros Desktop** - GUIの完全翻訳
* **Chloros Browser** - 全言語対応のWebインターフェース
* **Chloros CLI** - コマンドラインインターフェースおよび出力メッセージ
* **Chloros Python SDK** - APIメッセージおよびドキュメント

言語サポートにより、世界中のユーザーが母国語で障壁なく効率的に作業できます。
